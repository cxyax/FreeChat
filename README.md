# FreeChat
一个简易的无数据库JAVA语言聊天软件，有服务器端和客户端

编码格式为UTF-8

# 类介绍：
<strong>
ChatClient 客户端类<br/>
ChatServer 服务器端类<br/>
ClientReceiveImpl 客户端启动的接收线程<br/>
