# FreeChat
一个简易的无数据库JAVA语言聊天软件，有服务器端和客户端

编码格式为UTF-8

# 源码文件丢失问题
源码文件丢失，导致无法引入编译器,可以进入程序员阿鑫博客进行下载
相关文章链接：
http://www.cxyax.com/?post=254

# 类介绍：
ChatClient 客户端类<br/>
ChatServer 服务器端类<br/>
ClientReceiveImpl 客户端启动的接收线程<br/>
ServerFrame 服务器界面（可运行文件）<br/>
ClientFrame 客户端界面（可运行文件）<br/>
